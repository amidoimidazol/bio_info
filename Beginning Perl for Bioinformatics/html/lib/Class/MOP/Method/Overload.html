NAME

Class::MOP::Method::Overload - Method Meta Object for methods which implement overloading

VERSION

version 2.0604

DESCRIPTION

This is a Class::MOP::Method subclass which represents methods that implement overloading.

METHODS

Class::MOP::Method::Overload->wrap($metamethod, %options)

This is the constructor. The options accepted are identical to the ones accepted by Class::MOP::Method, except that it also required an operator parameter, which should be an operator as defined by the overload pragma.

$metamethod->operator

This returns the operator that was passed to new.
