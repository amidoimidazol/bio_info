NAME

PPI::Statement::When - Describes all compound statements

SYNOPSIS

  foreach ( qw/ foo bar baz / ) {
      when ( m/b/ ) {
          boing($_);
      }
      when ( m/f/ ) {
          boom($_);
      }
      default {
          tchak($_);
      }
  }

INHERITANCE

  PPI::Statement::When
  isa PPI::Statement
      isa PPI::Node
          isa PPI::Element

DESCRIPTION

PPI::Statement::When objects are used to describe when and default statements, as described in perlsyn.

METHODS

PPI::Structure::When has no methods beyond those provided by the standard PPI::Structure, PPI::Node and PPI::Element methods.

Got any ideas for methods? Submit a report to rt.cpan.org!
